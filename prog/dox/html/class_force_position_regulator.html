Detailed Description

Class defining Force/Position Regulator. In order to combine the features of stiffness control and force control, a parallel force/position regulator can be designed, where a PI force control action plus desired force feedforward is used in parallel to a PD position control action. Driving torques: u = J*( kp*(pos_d - p) + force_d + kf*(force_d - force) + ki*integral(force_d - force) ) - kv*vel
